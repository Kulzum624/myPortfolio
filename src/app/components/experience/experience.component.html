6-Month Internship

My professional journey in web development began with a six-month Front-End Development internship, where I strengthened my foundation in modern web technologies under the mentorship of experienced developers. Although I had already been building projects with React and Angular, working alongside senior engineers exposed me to industry-standard practices, advanced development techniques, and a deeper understanding of front-end architecture.

Starting with fundamental applications such as a Counter App, Blackjack Game, Rock-Paper-Scissors Game, and browser extensions, I gradually progressed to building responsive and interactive websites, including e-commerce platforms, restaurant websites, and a space tourism application that involved advanced CSS concepts and modern UI implementation.

As my expertise grew, I gained hands-on experience integrating RESTful APIs, handling asynchronous data, and working with real-world application workflows. One of the key milestones of my internship was contributing to a Hospital Management System built with React. This experience required a thorough understanding of API documentation, HTTP methods, Axios, authentication flows, and data management practices commonly used in production environments.

Beyond development, I became proficient in tools and technologies used across modern software teams, including Postman for API testing and debugging, Git for version control, and PostgreSQL for database management. Working on practical business applications enhanced my problem-solving abilities, attention to detail, and understanding of scalable software development.

This internship transformed me from a self-taught developer into a more confident and industry-ready engineer, capable of building modern, responsive, and user-focused web applications while collaborating effectively within professional development teams.
